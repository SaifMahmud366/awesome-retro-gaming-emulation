## 📝 Notes

- **ROM Legality**: Emulators are legal, but downloading ROMs for games you don't own may violate copyright laws. Always use legally obtained ROMs.
- **RetroArch Setup**: Download cores within RetroArch to emulate specific systems. Check [LibRetro Docs](https://docs.libretro.com/) for setup guides.
- **Discontinued Emulators**: Some emulators (e.g., Citra, yuzu) stopped official development in 2024, but community forks like Panda3DS or Sudachi may be active.

These operating systems turn your device into a retro gaming console, ideal for handhelds like Anbernic or Raspberry Pi.

| OS | Description |
|------------------------------------------------------|------------------------------------------------------------------------------------|

| 📦 [Recalbox](https://www.recalbox.com/) | A user-friendly retro gaming OS with support for multiple emulators. |
| 🥧 [RetroPie](https://retropie.org.uk/) | A customizable retro gaming platform for Raspberry Pi and other devices. |
